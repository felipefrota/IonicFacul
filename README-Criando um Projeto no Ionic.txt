PRE REQUISITOS
Instalar o nodejs

Intalar o Ionic com os codigos
npm install -g cordova ionic 
npm install -g cordova

CRIANDO O PROJETO
ionic start NomeProjeto tabs
-No local do tabs voce pode colocar blank, sidemenu. Todos mostram a estrutural do site https://ionicframework.com/getting-started


DPS DE CRIAR O PROJETO 
-Para criar a pasta WWW dentro do projeto e preciso entrar na pasta do projeto pelo cmd e dar o seguinte codigo
ionic build

-E para criar a pasta Config.xml e preciso entrar na pasta do projeto pelo cmd e dar o seguinte codigo 
ionic integrations enable cordova
